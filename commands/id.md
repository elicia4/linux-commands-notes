# `id`

[***Table of Contents***](/README.md)

Print user and group information for user(s):

```bash
id
```

```bash
id $USER random_name
```

To get the user ID (UID) of the (current by default) user:

```bash
id -u <user>
```

To get the username of the (current by default) user:

```bash
id -un <user>
```
